# ABOUT
Synthy is a simple FM synthesizer built using Rust. it is cross platform, but only compiles to VST2 right now.

# INSTRUCTIONS

to use synthy, simply:
1. run `cargo build` to generate a DLL file
2. move that generated DLL file into your system VST folder
3. have your DAW re-scan your VST folder
4. voila! you can now use synthy!
